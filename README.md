# projeto_analise_dados
Este é um projeto para meu portfólio de análise de dados utilizando o Jupyter. Trata-se de uma análise sobre uma loja de suplementos que busca auxiliar nas decisões importantes da empresa. A fim de aumentar sua lucratividade, esta análise destaca pontos relevantes, como despesas que podem ser reduzidas e produtos que merecem maior investimento.
